## Features

- TypeScript 2
- Jest
- TSLint ([airbnb config](https://www.npmjs.com/package/tslint-config-airbnb))
- Travis CI

## Getting started

```
npm run build
```

See [`package.json`](https://github.com/zurfyx/typescript-starter-kit/blob/master/package.json) for the rest of commands.
